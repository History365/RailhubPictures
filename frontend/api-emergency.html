Emergency Mode Information

What is Emergency Mode?

Emergency Mode is a special operating mode for the RailHub Pictures website that allows it to continue functioning even when the API is completely down.

Features in Emergency Mode:

  • Uses direct access to the main domain instead of API endpoints
  • Provides mock data for critical features when real data can't be accessed
  • Shows warning banner to users that the site is in emergency mode
  • Attempts to extract available images from the homepage for display
  • Maintains user sessions if possible

Limitations in Emergency Mode:

  • Limited to displaying pre-cached or mock data
  • New uploads may fail or be queued for later submission
  • User accounts may show limited or placeholder information
  • Search functionality may be limited or unavailable
