How to Use Dark Mode

  1. Click the moon/sun icon in the footer to toggle between light and dark modes.
  2. Your preference will be saved and remembered across visits.
  3. The theme will automatically match your system preference on first visit.

Implementation Details

The dark mode is implemented using CSS variables and a simple JavaScript toggle. The theme preference is stored in localStorage.

For developers: Check assets/stylesheet/theme.css and assets/scripts/footer.js to see how it's implemented.
